# ビルドとエラー修正

TypeScriptおよびビルドエラーを段階的に修正する:

1. ビルド実行: npm run build または pnpm build

2. エラー出力を解析:
   - ファイルごとにグループ化
   - 重要度順にソート

3. 各エラーに対して:
   - エラーのコンテキストを表示（前後5行）
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - ビルドを再実行
   - エラーが解決されたことを確認

4. 以下の場合は停止:
   - 修正が新しいエラーを生む
   - 3回試行しても同じエラーが残る
   - ユーザーが一時停止を要求

5. サマリーを表示:
   - 修正されたエラー
   - 残存エラー
   - 新しく発生したエラー

安全のため、エラーは一つずつ修正すること！
